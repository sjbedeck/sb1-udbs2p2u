## Overview

Smooth UI provides highly customizable, production-ready UI blocks for building beautiful websites and applications. Every component is designed with smooth animations and polished interactions.

## Features

- **Production Ready** – Battle-tested components ready for production
- **Smooth Animations** – Beautiful, fluid animations throughout
- **Customizable** – Highly customizable to match your brand
- **Block Library** – Complete UI blocks for rapid development
- **Modern Design** – Contemporary design patterns and aesthetics

## Usage

Copy and paste Smooth UI blocks into your project for instant beautiful, animated interfaces.
