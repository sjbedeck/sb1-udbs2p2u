## Overview

Plasma is a developer SaaS/app template designed specifically for developer tools and applications. It features a complete set of pages including landing, pricing, documentation, changelog, and download pages - everything you need to launch a developer-focused product.

Built with Next.js 15, shadcn/ui, and Tailwind 4, Plasma is available in both Next.js and Astro editions.

## Features

- **Developer-Focused**: Designed specifically for developer tools and apps
- **Complete Page Set**: Landing, pricing, docs, changelog, and download pages
- **Documentation Support**: Built-in documentation pages with MDX
- **Changelog Page**: Keep users updated with product changes
- **Download Page**: Perfect for distributing desktop apps or CLIs
- **Built with Next.js 15**: Latest performance and SEO features
- **Styled with Tailwind 4**: Easy customization
- **Dual Framework Support**: Available in Next.js and Astro editions
- **shadcn/ui Components**: Consistent component library
- **Figma Files Included**: Design files for customization
