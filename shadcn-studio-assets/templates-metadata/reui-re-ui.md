## Overview

Re-UI is an open-source collection of UI components and animated effects built with React, TypeScript, Tailwind CSS, and Motion. It pairs beautifully with shadcn/ui to enhance your applications with stunning animations.

## Features

- **Animated Components** – Beautiful animations powered by Motion
- **TypeScript Support** – Full TypeScript support for type safety
- **Tailwind CSS** – Styled with Tailwind for easy customization
- **shadcn/ui Compatible** – Works seamlessly with shadcn/ui
- **Open Source** – Free to use and contribute

## Usage

Browse the Re-UI component library and copy the animated components you need into your project.
