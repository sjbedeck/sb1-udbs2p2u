## Overview

Ruixen UI is a lightweight and customizable React component library built on top of shadcn/ui. It provides a collection of beautifully designed components that are easy to use and customize.

## Features

- **Lightweight** – Minimal bundle size for optimal performance
- **Customizable** – Easy to customize to match your brand
- **React-Based** – Built specifically for React applications
- **Modern Design** – Clean and modern component designs
- **Open Source** – Free to use and modify

## Usage

Install Ruixen UI components and import them into your React project for instant beautiful interfaces.
