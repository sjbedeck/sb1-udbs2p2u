## Overview

Zippay is a modern fintech SaaS and marketing template that offers a robust multi-page design specifically crafted for fintech SaaS and marketing websites. Its professional design conveys trust and reliability essential for financial products.

Built with Next.js 15, shadcn/ui, and Tailwind 4, Zippay is available in both Next.js and Astro editions.

## Features

- **Fintech-Focused Design**: Professional aesthetic tailored for financial services
- **Multi-Page Template**: Complete set of pages for SaaS marketing
- **Built with Next.js 15**: Optimized performance and SEO capabilities
- **Styled with Tailwind 4**: Flexible customization options
- **Dual Framework Support**: Available in both Next.js and Astro editions
- **Responsive Layout**: Perfect display on all devices
- **shadcn/ui Components**: Accessible and customizable components
- **Figma Files Included**: Original design files included
