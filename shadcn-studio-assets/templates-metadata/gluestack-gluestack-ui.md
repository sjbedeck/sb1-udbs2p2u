## Overview

Gluestack UI provides React & React Native components styled with Tailwind CSS. It offers universal components that work seamlessly across web and mobile platforms, making it ideal for cross-platform development.

## Features

- **Universal Components** – Works on both web and mobile
- **React Native Support** – Full React Native compatibility
- **Tailwind CSS** – Styled with Tailwind for consistency
- **Accessible** – Built with accessibility in mind
- **Customizable** – Easy to theme and customize

## Usage

Install Gluestack UI and use the same components across your React web and React Native mobile applications.
