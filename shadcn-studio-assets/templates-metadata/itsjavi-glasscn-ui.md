## Overview

Glasscn UI is a shadcn/ui component library featuring glassmorphism variants and many additional components. It brings beautiful frosted glass effects to your applications while maintaining full compatibility with shadcn/ui.

## Features

- **Glassmorphism Design** – Beautiful frosted glass effects
- **Additional Components** – Extended component library beyond shadcn/ui
- **shadcn/ui Compatible** – Drop-in replacement for shadcn/ui components
- **Modern Aesthetic** – Trendy glass-like design patterns
- **Open Source** – Free to use and customize

## Usage

Replace or extend your shadcn/ui components with Glasscn UI variants for a modern glassmorphism look.
