## Overview

Commerce UI provides components, blocks, and examples specifically designed for building e-commerce storefronts and applications. It includes everything you need to create beautiful and functional online stores.

## Features

- **E-commerce Focused** – Components designed for online stores
- **Product Cards** – Beautiful product display components
- **Cart & Checkout** – Pre-built cart and checkout flows
- **Category Pages** – Ready-to-use category layouts
- **Built on shadcn/ui** – Consistent with shadcn/ui design

## Usage

Browse the Commerce UI library and copy the e-commerce components you need to build your online store.
