## Overview

Chanhdai.com is a minimal portfolio template that combines a personal website, component registry, and blog. It showcases how to build a beautiful developer portfolio with shadcn/ui.

## Features

- **Minimal Design** – Clean and focused aesthetic
- **Component Registry** – Built-in component showcase
- **Blog Integration** – Integrated blog functionality
- **Portfolio** – Personal portfolio section
- **Open Source** – Free to use as a template

## Usage

Fork the repository and customize it to create your own portfolio, blog, and component registry.
