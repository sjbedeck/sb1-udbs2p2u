## Overview

mapcn provides beautiful, ready-to-use map components for React applications. Built on MapLibre and styled with Tailwind CSS, it integrates seamlessly with shadcn/ui projects.

## Features

- **MapLibre Based** – Built on the powerful MapLibre mapping library
- **Tailwind Styled** – Consistent styling with Tailwind CSS
- **shadcn Registry** – Install via shadcn CLI
- **Customizable** – Fully customizable map components
- **React Components** – Easy to integrate into React projects

## Installation

Install map components directly via the shadcn CLI:

```bash
npx shadcn@latest add https://mapcn.dev/maps/map.json
```

## Usage

Browse the mapcn documentation to see available map components and add them to your project.
