## Overview

ElevenLabs UI is a collection of open source agent and audio components designed for building voice-enabled AI applications. It includes beautiful visualizations and interactive components for audio playback, voice chat, and AI agent interfaces.

## Features

- **Agent Orbs** – Interactive orb visualization with agent states (idle, listening, talking)
- **Waveform** – Real-time audio visualization with smooth scrolling animation
- **Voice Chat** – Customer support voice chat components
- **Audio Player** – Full-featured audio player with controls
- **Voice Fill** – Voice-enabled form filling powered by ElevenLabs Scribe
- **Conversation UI** – Chat interface with voice and text support

## Components

- Agent orbs with multiple states
- Real-time waveform visualizations
- Audio player with playback controls
- Voice chat interfaces
- Conversation components with voice and text

Browse the ElevenLabs UI documentation to see available components and add them to your voice-enabled AI project.
