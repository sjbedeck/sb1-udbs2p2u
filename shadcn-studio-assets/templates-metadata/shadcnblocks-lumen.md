## Overview

Lumen is a luminescent, modern landing page template that contains all the core pages and sections needed for a marketing website. Its bright and airy design creates an inviting atmosphere for visitors.

Built with Next.js 15, shadcn/ui, and Tailwind 4, Lumen is available in both Next.js and Astro editions, providing flexibility for your preferred development stack.

## Features

- **Luminescent Design**: Bright, modern aesthetic that stands out
- **Complete Page Set**: All core pages and sections for a marketing website
- **Built with Next.js 15**: Latest Next.js features for performance and SEO
- **Styled with Tailwind 4**: Rapid customization with utility-first CSS
- **Dual Framework Support**: Available in both Next.js and Astro editions
- **Responsive Layout**: Optimized for all screen sizes
- **shadcn/ui Components**: Consistent, accessible component library
- **Figma Files Included**: Design files for further customization
