## Overview

Shuip provides ready-to-use, business-focused components that help you ship faster. It's designed to accelerate development by offering pre-built UI components commonly needed in business applications.

## Features

- **Business-Focused** – Components designed for real-world business use cases
- **Ship Faster** – Pre-built components reduce development time
- **Production Ready** – Battle-tested components ready for production
- **Built on shadcn/ui** – Consistent styling with shadcn/ui
- **Well Documented** – Comprehensive documentation for easy implementation

## Usage

Visit the Shuip documentation to browse available components and copy them into your project.
