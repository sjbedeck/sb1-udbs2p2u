## Overview

Shaduxe UI provides additional component variants for shadcn/ui. It extends the base shadcn/ui components with new variants, styles, and customization options.

## Features

- **Extended Variants** – Additional variants for shadcn/ui components
- **Drop-in Replacement** – Works seamlessly with existing shadcn/ui setup
- **Customizable** – Easy to customize and extend
- **Consistent Design** – Maintains shadcn/ui design language
- **Open Source** – Free to use and contribute

## Usage

Browse the Shaduxe UI component library and add the variants you need to your shadcn/ui project.
