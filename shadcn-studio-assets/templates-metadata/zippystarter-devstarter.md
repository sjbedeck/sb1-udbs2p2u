## Overview

Devstarter is a free, open-source one-page portfolio template designed for developers who want to stand out. Built with Next.js, shadcn/ui, and Tailwind CSS, it features a distinctive cyberpunk aesthetic with neon green accents against dark backgrounds.

## Features

- **One-page portfolio** – Complete portfolio with hero, projects, skills, blog, and contact sections
- **Cyberpunk aesthetic** – Bold neon accents, monospace typography, and futuristic UI elements
- **Dark & light modes** – Both themes fully styled and ready to use
- **shadcn theming support** – Built on shadcn/ui for easy customization of colors, fonts, and components
- **Fully responsive** – Looks great on all devices from mobile to ultrawide monitors
- **Modern CSS** – Uses CSS Grid, Subgrid, custom properties, backdrop blur, and blend modes

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- shadcn/ui
- TypeScript
