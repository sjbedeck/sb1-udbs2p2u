## Overview

Metafi is a modern template designed for creating sophisticated marketing sites. Built with the latest technologies including Next.js 15, shadcn/ui, and Tailwind 4, it provides a polished foundation for professional web presence.

Available in both Next.js and Astro editions, Metafi offers flexibility in choosing your preferred framework while maintaining consistent design quality.

## Features

- **Modern Design**: Sophisticated and contemporary aesthetic perfect for marketing sites
- **Built with Next.js 15**: Leverages the latest Next.js features for optimal performance
- **Styled with Tailwind 4**: Utility-first CSS framework for rapid customization
- **Dual Framework Support**: Available in both Next.js and Astro editions
- **Responsive Layout**: Fully responsive design for all devices
- **shadcn/ui Components**: Built on the popular shadcn/ui component library
- **Figma Files Included**: Original design files included for customization
- **TypeScript Support**: Full TypeScript support for type safety
